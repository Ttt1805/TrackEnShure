Сотрудники: преподаватель, лаборант, студент.

Создать метод «Выполнить работу». 
Соответственно для каждого это будет :
- Преподаватель - провести занятие (лекцию, практическое занятие) и т.д.;
- Лаборант - отремонтировать компьютер, установить программное обеспечение и т.д.;
- Студент - посетить лекцию, выполнить д.з. и т.д.


Подумать куда можно расширить пример фундаментальными паттернами (помимо очевидных: интерфейс, функциональный дизайн).
